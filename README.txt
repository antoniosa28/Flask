# 🛠️ Ferramenta de Correção de Bugs com IA — TCC

Este é o repositório do meu Trabalho de Conclusão de Curso (TCC), que apresenta uma ferramenta web desenvolvida em **Python** com *Flask*, focada no **auxílio à correção de erros de programação**. A aplicação integra a **tecnologia de linguagem natural (LLM)** do Google Generative AI (Gemini) para fornecer explicações detalhadas sobre erros em códigos-fonte.

## 📌 Objetivo

Desenvolver uma ferramenta que auxilie programadores a identificarem e compreenderem os erros de seus códigos de forma clara e didática, utilizando **Inteligência Artificial Generativa** como suporte.

## 🚀 Funcionalidades

- 🔍 Análise automática de código-fonte enviado pelo usuário.
- 🧠 Interpretação e explicação de erros com base na IA generativa.
- 💬 Respostas explicativas baseadas em linguagens como:
  - Python
  - C
  - Java
  - Kotlin
- 🌐 Interface web leve e responsiva (HTML + CSS + JavaScript).
- ☁️ Hospedagem na nuvem via [PythonAnywhere](https://www.pythonanywhere.com/).

## 🛠️ Tecnologias Utilizadas

- **Backend:**
  - [Python](https://www.python.org/)
  - [Flask](https://flask.palletsprojects.com/)
  - Integração com API do [Google Gemini](https://ai.google.dev/)
- **Frontend:**
  - HTML5, CSS3, JavaScript (sem frameworks pesados)
- **Outros:**
  - JSON para comunicação assíncrona (AJAX)
  - Git para versionamento de código
